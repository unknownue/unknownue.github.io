## 本次PR的故事

这个PR的目标很明确：填补几何模块中API的一个小缺口。在Bevy的数学库中，`Measured2d` trait 为二维几何图形定义了计算面积和周长的方法。它是一个实用工具，使得不同形状可以共享统一的测量接口。然而，并非所有适合拥有此trait的图形都实现了它，这导致了API的不一致，并迫使开发者在需要时自行计算。

问题主要集中在两个图形上：三维空间中的平面（`Plane3d`）和二维的环形（`Ring`）。`Plane3d` 虽然是一个三维对象，但它通常被概念化为一个二维平面，拥有长度和宽度（`half_size`）。因此，为其计算局部二维空间中的“面积”和“周长”是有意义的，这也延续了先前为 `Triangle3d` 设定的先例。至于 `Ring`，它是一个由内外两个轮廓定义的复合图形，其测量值自然应由其组成部分决定。

解决方案非常直接，体现了良好的工程实践：为这两个缺失的类型添加 `Measured2d` trait 的实现。对于 `Ring<P>`，其实现利用了泛型约束 `P: Primitive2d + Measured2d`，这确保了构成环形的内外图形（`outer_shape` 和 `inner_shape`）自身已经实现了测量方法。环形的面积被定义为外轮廓面积减去内轮廓面积，这符合环形是“挖空”图形的直观理解。其周长则是内外轮廓周长之和，这在几何上是正确的。

```rust
// Ring<P> 的 Measured2d 实现
impl<P: Primitive2d + Measured2d> Measured2d for Ring<P> {
    #[inline]
    fn area(&self) -> f32 {
        self.outer_shape.area() - self.inner_shape.area()
    }

    #[inline]
    fn perimeter(&self) -> f32 {
        self.outer_shape.perimeter() + self.inner_shape.perimeter()
    }
}
```

对于 `Plane3d`，其测量计算基于其 `half_size` 属性，即平面在其局部X轴和Y轴上的半边长。面积计算为 `(half_size.x * 2) * (half_size.y * 2)`，等价于代码中的 `half_size.element_product() * 4.0`。周长则是各边长度之和 `2 * (half_size.x * 2 + half_size.y * 2)`，即 `half_size.element_sum() * 4.0`。

```rust
// Plane3d 的 Measured2d 实现
impl Measured2d for Plane3d {
    #[inline]
    fn area(&self) -> f32 {
        self.half_size.element_product() * 4.0
    }

    #[inline]
    fn perimeter(&self) -> f32 {
        self.half_size.element_sum() * 4.0
    }
}
```

这两个实现都使用了 `#[inline]` 属性。这是一个合理的优化选择，因为这些计算非常简单（基本是算术操作），内联可以避免函数调用的开销，提升性能，特别是考虑到它们可能在渲染或物理循环中被频繁调用。

从技术角度来看，这个PR没有引入任何破坏性变更，它纯粹是API的扩展。它完善了Bevy几何图元的功能集，使得像`Plane3d`和`Ring`这样的图形现在可以无缝地与其他已经实现`Measured2d`的图形（如`Rectangle`、`Circle`、`Triangle2d`）一起使用。这对于需要根据图形物理属性（如面积用于质量计算）或渲染属性（如周长用于描边效果）进行通用处理的系统代码尤其有用。这次修改也保持了代码库的整洁性和一致性，遵循了“为所有合适的东西实现相关trait”的原则。

## 关键文件更改

### 1. `crates/bevy_math/src/primitives/dim2.rs` (+12/-0)
**修改内容与原因**：为 `Ring<P>` 结构体添加了 `Measured2d` trait 的实现。这使得环形能够提供面积和周长计算，完善了其功能。

**关键代码片段**：
```rust
impl<P: Primitive2d + Measured2d> Measured2d for Ring<P> {
    #[inline]
    fn area(&self) -> f32 {
        self.outer_shape.area() - self.inner_shape.area()
    }

    #[inline]
    fn perimeter(&self) -> f32 {
        self.outer_shape.perimeter() + self.inner_shape.perimeter()
    }
}
```

### 2. `crates/bevy_math/src/primitives/dim3.rs` (+11/-0)
**修改内容与原因**：为 `Plane3d` 结构体添加了 `Measured2d` trait 的实现。虽然是一个3D图元，但根据其二维局部坐标系提供测量值是合理且有先例的（如 `Triangle3d`）。

**关键代码片段**：
```rust
impl Measured2d for Plane3d {
    #[inline]
    fn area(&self) -> f32 {
        self.half_size.element_product() * 4.0
```
