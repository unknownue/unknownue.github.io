1. **性能改进**：Node.js 24.x 包含了 V8 引擎的更新版本，提供了更好的性能和内存管理

3. **新特性支持**：支持最新的 JavaScript 语言特性和 API
4. **兼容性要求**：需要 GitHub Actions 运行器版本至少为 v2.327.1
